drop_while()

drop_while(\Psl\Iter\iterable  $iterable, callable  $predicate)

Drops items from an iterable until the predicate fails for the first time.

This means that all elements after (and including) the first element on which the predicate fails will be yield.

Examples:

 Iter\drop_while([3, 1, 4, -1, 5], fn($i) => $i > 0)
 => Iter(-1, 5)
