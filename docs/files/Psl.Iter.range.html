range()

range(  $start,   $end,   $step = null)

Creates an iterator containing all numbers between the start and end value (inclusive) with a certain step.

Examples:

Iter\range(0, 5)
=> Iter(0, 1, 2, 3, 4, 5)

Iter\range(5, 0)
=> Iter(5, 4, 3, 2, 1, 0)

Iter\range(0.0, 3.0, 0.5)
=> Iter(0.0, 0.5, 1.0, 1.5, 2.0, 2.5, 3.0)

Iter\range(3.0, 0.0, -0.5)
=> Iter(3.0, 2.5, 2.0, 1.5, 1.0, 0.5, 0.0)

Parameters

$start
$end
$step

Throws

\Psl\Exception\InvariantViolationException

If $start < $end, and $step is negative.
