Functions

reduce_with_keys()

reduce_with_keys(\Psl\Iter\iterable  $iterable, callable  $function,   $initial = null)

Reduce iterable with both values and keys using a function.

The reduction function is passed an accumulator value and the current iterator value and returns a new accumulator. The accumulator is initialized to $initial.

Examples:

 Iter\reduce_with_keys(
     Iter\range(1, 5),
     static fn(int $accumulator, int $key, int $value): int => $accumulator + $value,
     0,
)
 => 15

 Iter\reduce_with_keys(
     Iter\range(1, 5),
     static fn(int $accumulator, int $key, int $value): int => $accumulator * $value,
     0,
)
 => 120
