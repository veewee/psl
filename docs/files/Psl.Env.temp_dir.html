temp_dir()

Returns the value of the "TMPDIR" environment variable if it is set, otherwise it returns /tmp.

Tags

package

Default
note

On windows, we can't count on the environment variables "TEMP" or "TMP", and so must make the Win32 API call to get the default directory for temporary files.

The return value of this function can be overridden using the sys_temp_dir ini directive.
